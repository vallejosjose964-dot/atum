Entrada macro

Lee los archivos canónicos del repo y manda el archivo completo al backend. El frontend no recompone curvas ni recalcula resultados.

Configuración
Solo cámbialo si moviste el backend. El frontend reintenta automáticamente antes de declarar caída.
Assets requeridos al lado de este HTML: manifest.json, galaxies_batch.csv y clusters.csv. El manifest debe apuntar a esos mismos archivos.

Entrada micro

Micro es directo. Solo manda la partícula al endpoint /micro y dibuja la respuesta del backend.

Endpoint micro fijado: /micro
JSON enviado: { "particle": "W | Z | X | MUON" }. Sin derivaciones locales.
