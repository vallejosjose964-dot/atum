INSTRUCCIONES

ATUM es una consola científica para ejecutar cálculos físicos mediante el motor matemático THOT.

• El frontend NO realiza cálculos físicos.
• Todos los resultados son calculados por el backend THOT en tiempo real.
• El modelo físico utiliza una única constante fundamental τ.
